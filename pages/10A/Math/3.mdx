# 不等式

## 比大小

### 作差法

- $a - b > 0 \Rightarrow a > b$.
- $a - b = 0 \Rightarrow a = b$.
- $a - b < 0 \Rightarrow a < b$.

### 作商法

若 $a, b > 0$ 且 $\frac{a}{b} > 1$，则 $a > b$.

### 乘方

例题：求 $2^{ \frac{2}{5} }$ 和 $3^{ \frac{1}{3} }$ 大小关系.

$5$ 和 $3$ 最小公倍数是 $15$，同乘 $15$ 次方：

$2^{ 15 \frac{2}{5} } = 2^6 = 64 < 3^{ 15 \frac{1}{3} } = 3^5 = 243$，

$2^{ \frac{2}{5} } < 3^{ \frac{1}{3} }$.

## 不等式的性质

- 对称性： $a > b \Leftrightarrow b < a$.
- 传递性：$a > b, b > c \Rightarrow a > c$.
- 加法原则：$a > b \Leftrightarrow a + c > b + c$.
- 乘法原则：$a > b, c > 0 \Rightarrow ac > bc$；$a > b, c < 0 \Rightarrow ac < bc$.
- $a > b, c > d \Rightarrow a + c > b + d$.
- $a > b > 0, c > d > 0 \Rightarrow ac > bd$.
- $a > b > 0 \Rightarrow a^n > b^n ( n \in \mathbb{N}^{\ast} )$.

## 不等式公式

重要不等式：$a^2 + b^2 \geq 2ab, a = b$ 时取等. 可通过 $( a - b )^2 = a^2 - 2ab + b^2 \geq 0$ 推导.

基本不等式：$a, b > 0, a + b \geq 2 \sqrt{ab}$，将重要不等式 $a, b$ 替换为 $\sqrt{a}, \sqrt{b}$ 即可.
